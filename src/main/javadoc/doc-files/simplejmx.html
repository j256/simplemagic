SimpleJMX

Version 1.1 - February 2013

This package provides some classes that publish object using Java's Management Extensions (JMX). These published objects can be investigated with jconsole (Sun's graphical JMX user interface application) or another JMX client. Included is also a programmatic JMX client which you can use to connect to and interrogate remote JMX servers.

To get started quickly using SimpleJMX, see section Start Using Quickly. You can also take a look at the the examples section of the document which has various working code packages. See section Example Code. There is also a PDF version of this documentation.
